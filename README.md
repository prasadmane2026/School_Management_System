# School Management System (Core Java)

A console-based School Management System built using Core Java.
This project is created for practicing Object-Oriented Programming concepts, CRUD operations, Collections Framework, File Handling, and Git/GitHub workflow.

## Features

### Student Management

* Add Student
* View All Students
* Search Student by ID
* Update Student School Details
* Update Student Personal Details
* Delete Student
* Prevent Duplicate Student IDs

### Teacher Management

* Add Teacher
* View All Teachers
* Search Teacher by ID
* Update Teacher Office Details
* Update Teacher Personal Details
* Delete Teacher
* Prevent Duplicate Teacher IDs

### File Handling

* Store student records permanently in `students.txt`
* Load student data automatically when application starts

1. Clone the repository
2. Open project in IntelliJ IDEA
3. Run `Main.java`
